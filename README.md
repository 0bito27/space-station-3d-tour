#  Space Station 3D Tour

Interactive 3D web application that simulates a virtual tour of a space station using Three.js.

##  Features

*  Interactive 3D space station
*  Rotation of solar panels
*  Animated antenna signal
*  Satellite orbiting around the station
*  Clickable objects with information panel
*  Multiple camera views (station, antenna, panels, satellite)
*  Station rotation mode with UI controls
*  Exit mode and UI state management

##  Technologies Used

* Three.js
* Vite
* JavaScript (ES6)
* HTML5 / CSS3

##  Controls

* Use UI buttons to navigate between different parts of the station
* Click on objects to display additional information
* Activate rotation mode for dynamic view
* Follow the satellite for a moving camera perspective

## Setup

Due to GitHub file size limitations, the 3D model is not included in the repository.

### Steps to run:

1. Download the model:
   https://github.com/0bito27/space-station-3d-tour/releases/download/v1.0/sci_fi_space_station.glb

2. Place it in:
   `/public/models/`

3. Run the project.

⚠️ Without this file, the 3D scene will not load the main space station!

##  Installation

```bash
npm install
npm run dev
```
